For ease of vote event creation

You have the power to create and modify event, determine the period of event, choose participants, decide the result's release.
  • Click 'Create New Vote Event" in Overview section
  • Fill up the neccessary details. Make sure the details are correct and not outdated
  • Decide the participants and fill them up in .csv file with the format of Name, Email. Download the template in vote event creation section
  • If you miss out anything, you can go back to modify the event you created by clicking "Modify" in Overview section
  • Click "Confirm" to publish the event so the voters can take part during the start date and time of the event
  • By confirming, you no longer have the power to modify the event details except the end date and time
Click 'View' in Overview section. From there, you can view anything about the details
Click 'Delete' in Overview section. Then, confirm your decision. Voila, the event is deleted
