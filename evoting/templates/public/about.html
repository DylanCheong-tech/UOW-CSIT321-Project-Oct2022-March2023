About HARPOCryption

A voting web application that provides platform for users for the purpose of confidentiality, integrity and availability by performing homomorphic encryption process on encrypted votes and provide secure routes and anonymity to the voters. This web app is developed by:

Overview of HARPOCryption system

To create a vote event or election, an administrator needs an account. To register for an account, please click 'Get Started'. Login with your account and create your first vote event there. Any helps regarding about administrator's functionalities, please refer to User Guide in the administrator portal. You have the most power regarding about the vote event, please use it wisely.
To vote, you need to have the authorisation token. You will receive the token via email if you are invited to an vote event. The unique authorisation token is created by our system. In order to have maximum security as a voter, please avoid sharing your authorisation token or link in the email.
Once you submit your vote, you cannot longer vote. This is to prevent double voting. Vote booth link with authorisation token example: 'harpocryption/voter/vote?auth={token}'.
To check final results of the vote event, please wait for the administrator's decision to release the results. After the result is released, you can use your new authorisation token via email to view the result booth. Example of the link with the authorisation token: 'harpocryption/voter/finalresult?auth={token}.
